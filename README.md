# 🍿 FE_DUANTOTNGHIEP_BANVEXEMPHIM (REACT.JS + VITE)

Dự án này là Frontend cho hệ thống bán vé xem phim, kết nối với Spring Boot qua REST API.

## 🚀 CÁCH CHẠY DỰ ÁN
1. **Mở Terminal:** Tại thư mục này.
2. **Cài đặt thư viện:** `npm install` (nếu là lần đầu).
3. **Chạy dự án:** `npm run dev`.
4. **Trình duyệt:** `http://localhost:5173`.

## 📦 THƯ VIỆN ĐÃ TÍCH HỢP
- **React 19**, **Vite**, **Axios**, **React Router Dom**, **Lucide React**, **@dnd-kit** (Kéo thả ghế).

## 📁 CẤU TRÚC THƯ MỤC CHÍNH
- `src/pages`: Giao diện (Home, Login, Admin).
- `src/components`: Thành phần tái sử dụng.
- `src/services`: Gọi API qua Axios.
- `src/routes`: Quản lý đường dẫn và bảo mật.
